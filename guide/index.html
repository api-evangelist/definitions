My Industry Guide To API Definitions

I research many areas of the API space, and I've carved off a specific project to keep track of what is happening with API specifications, definitions, schema, and scopes. I publish all of this research regularly to a Github repository, within the API Evangelist organization.

When I can, I take each of my top reserch areas and distill them down into a single industry guide, providing a single portable snapshot of the world of API definitions--one that is updated regularly, keeping pace with what is going on.

I am currently evolving the layout and process for my API industry guides. I have a draft copy of my API definition industry guide available in the Github repository for this area of my research. You can still get a copy of the previous API definition guide, but you can also get at the draft of the next version as well.

Each of my API industry guides include opportunities for sponsored content, as well as possessing extra sponsorship slots for my partners, and folks who help me edit each guide--if you'd like to get involved you can email info@apievangelist.com.
